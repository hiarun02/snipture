# Code of Conduct

## Our Pledge

We want this project to be a welcoming, safe, and harassment-free experience for everyone. We value respectful collaboration and constructive feedback.

## Expected Behavior

- Be respectful, inclusive, and kind.
- Assume good intent and communicate clearly.
- Provide constructive, actionable feedback.
- Respect differing viewpoints and experiences.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks.
- Trolling, bullying, or sustained disruption of discussions.
- Sexualized language or imagery.
- Doxxing, threats, or intimidation.

## Enforcement

If you witness or experience unacceptable behavior, please report it to the maintainers by opening a private report or contacting the repository owner directly. The maintainers will review and take appropriate action, which may include warnings or removal from the community spaces.

## Scope

This Code of Conduct applies to all project spaces, including issues, pull requests, discussions, and any community channels related to this repository.

## Acknowledgment

By participating, you agree to follow this Code of Conduct.
